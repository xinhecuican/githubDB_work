# 23合并 *k* 个链表

### 题目描述

  合并 *k* 个排序链表，返回合并后的排序链表。请分析和描述算法的复杂度。 

 **示例:**

```
输入:
[
  1->4->5,
  1->3->4,
  2->6
]
输出: 1->1->2->3->4->4->5->6输入: 1->2
输出: false
```

### 知识点：

- **链表**

- **堆**


### 思路：

​	**1.0版**：**堆**

​	**2.0版**：**堆+自定义比较器**

​	**3.0版:** **分治**

**1.0版：**

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */ 
class Solution {
	public ListNode mergeKLists(ListNode[] lists) {
        PriorityQueue<Integer> queue = new PriorityQueue<>();
        ListNode dummyHead = new ListNode(0);//虚拟头结点
        ListNode temp = new ListNode(0) ,temp2 ;
        for (int i = 0; i < lists.length; i++) {//遍历链表 添加结点
            while (lists[i]!=null)
            {
                queue.add(lists[i].val);
                lists[i]=lists[i].next;
            }
        }
        if(queue.size()>0)
        {
            temp = new ListNode(queue.poll());
            dummyHead.next = temp;
        }

        while (queue.size()>0)
        {
            temp2=new ListNode(queue.poll());
            temp.next = temp2;
            temp = temp2;
        }
        return dummyHead.next;
    }//end method
}

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    //自定义比较器
    public Comparator<ListNode> ListNodeC = new Comparator<ListNode>(){
        @Override
        public int compare(ListNode l1, ListNode l2) {
            return l1.val-l2.val ;
        }
    };

    public ListNode mergeKLists(ListNode[] lists) {
        PriorityQueue<ListNode> queue = new PriorityQueue<ListNode>(ListNodeC);
        ListNode dummyHead = new ListNode(0);//虚拟头结点
        ListNode temp = new ListNode(0) ,temp2 ;
        for (int i = 0; i < lists.length; i++) {//遍历链表 添加结点
            while (lists[i]!=null)
            {
                queue.add(lists[i]);
                lists[i]=lists[i].next;
            }
        }

        if(queue.size()>0)
        {
            temp = queue.poll();
            dummyHead.next = temp;
        }

        while (queue.size()>0)
        {
            temp2= queue.poll();
            temp.next = temp2;
            temp = temp2;
        }
        return dummyHead.next;
    }//end method
}
